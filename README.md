# Aurum Decor API

REST API built with Node.js and Express, serving product data for the Aurum Decor Catalog frontend.

## Endpoints
- GET /api/products — list all products
- GET /api/products/:id — get a single product
- POST /api/products — add a new product
- DELETE /api/products/:id — remove a product

## Tech Stack
Node.js · Express · CORS

Frontend: https://github.com/HamzyCode/aurum-decor-app
